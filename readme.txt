Create a new page, in the article text input "[readers]", issued after will do. Through the short code form your recent in your blog readers to speak out of the list, and add the background set items, can set the number of readers wall and when the mouse pointer up the word when body color, so that you do not need to go to modify code. My-Readers-Wall 通过短代码的形式把你最近在你博客上发言的读者列表出来,并添加了后台设置项，可以设置读者墙的数量及当鼠标移上去时的字体颜色，这样就不需要您去修改代码.在后台操作就可以了.

= Basic Usage =

Create a new page, in the article text input "[readers]"

在后台新建一个页面 命名随意，文章属性保持默认;之后在文章正文部分输入 “[readers]” 发布就行了.

== Installation ==

1. My-Readers-Wall.zip to your Wordpress plugins directory, usually `wp-content/plugins/` and unzip the file.  It will create a `wp-content/plugins/My-Readers-Wall/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Create a new page, in the article text input "[readers]", issued after will do.
